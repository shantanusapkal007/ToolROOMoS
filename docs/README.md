# ToolRoom ERP – Documentation Suite

Welcome to the official documentation for ToolRoom ERP. This directory contains the complete enterprise documentation suite, broken down by role and technical domain.

Please refer to the appropriate guide below:

## 1. End Users & Operations
*   [**Manufacturing Process Guide**](manufacturing_process.md): Explains the deep workflow spanning Engineering (BOMs, ECOs), Production (Work Orders, MES), Inventory (Traceability, WMS), Quality, and Financial Costing.
*   [**Role-Based Operator Manuals**](operator_manuals.md): Focused, task-based guides strictly outlining the daily operations for specific roles (Sales, Procurement, Warehouse, Machine Operators, Quality, Finance).
*   [**Troubleshooting & Best Practices**](troubleshooting_best_practices.md): Solutions for common errors, hardware faults (barcode scanners, offline machines), and structural best practices (naming conventions, warehouse layouts).

## 2. Administration & Technical Reference
*   [**Architecture & Technical Reference**](architecture_guide.md): Details the advanced patterns powering the ERP, including CQRS, Event-Driven Architecture, Outbox Patterns, Idempotency, and the Truth Engine.
*   [**Administrator Guide**](administrator_guide.md): Instructions for system administrators configuring the organizational hierarchy (Tenants, Companies, Plants), Security Policies, Approval Workflows, and System Settings.
*   [**Developer & API Guide**](developer_api_guide.md): API specifications, authentication, Webhooks, error codes, and the underlying PostgreSQL database schema relationships.
